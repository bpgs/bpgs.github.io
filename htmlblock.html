Demo Seite mit HTML Block

Um mit einem HTML-Block zu arbeiten, muss man die Erweiterung HTML Block installieren. Diese Erweiterung stammt nicht vom Anbieter des Mobirise Systems sondern von DeltaPi.

Kurzbeschreibung HTML Block und gleichzeitig Beispiel für einen langen Text in einem HTML Block.

Ein HTML Block hat nur eine bestimmte Breite, obwohl das Block-Element sich bei der Bearbeitung über die gesamte Breite erstreckt. Allerdings wird der angezeigte Text auch immer breiter, wenn man mehr Text eingibt. Zum Vergleich füge ich weiter unten noch einen HTML-Block mit wenig Text ein. Man wird dann sehen, dass bei diesem die Breite deutlich geringer ist.

So ein HTML-Block hat noch einen weiteren Nachteil: Bei der publizierten Webseite taucht nach einem Klick auf den HTML Block eine Art Online HTML Editor auf. Das erscheint mir volkkommen sinnlos. Diesen Editor könnte man allenfalls beim Bearbeiten der Seite direkt in Mobirise gebrauchen. Ich habe gerade festgestellt, dass dieser Editor auch direkt in Mobirise verfügbar ist.
