Eine Mobireise-Erweiterung hat den Dateinamen erweiterungsname.mbrext. Tatsächlich handelt es sich aber um eine .zip-Datei, also um ein geacktes Archiv. Man kann so eine Datei z.B. umbenennen in erweiterungsname.mbrext.zip und dann entpacken. Dann findet man die folgenden Bestandteile (entweder im Hauptverzeichnis ode in einem Unterverzeichnis, dass nach dem Namen der Erweiterung benannt ist):
  • Datei params.json, enthält
  • Datei splash.html, enthält
  • Datei thumb.png, ist nicht immer vorhanden, wird ggf. in Datei splash.html referenziert, enthält eine Art Vorschaubild für die Installation.
  • Datei thumbnail.png, enthält
  • Datei erweiterungsname.gif, ist nicht immer vorhanden, wird ggf. in Datei splash.html referenziert, enthält eine Art Vorschaubild für die Installation.
  • Datei unterverzeichnis/component.js, enthält
  • Datei unterverzeichnis/template.html, enthält 
  • Datei unterverzeichnis/thumb.png, enthält
  • Datei unterverzeichnis/assets/*
  • Datei unterverzeichnis/
